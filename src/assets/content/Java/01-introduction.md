# Introduction à Java

Java est un langage de programmation orienté objet, développé par Sun Microsystems (maintenant Oracle) et publié pour la première fois en 1995. Java est conçu pour être portable, sécurisé et performant, avec le principe "Write Once, Run Anywhere" (WORA).

## Pourquoi Java ?

### Avantages principaux

- **Portabilité** : Le code Java compile en bytecode qui s'exécute sur la JVM, permettant l'exécution sur n'importe quelle plateforme
- **Robustesse** : Gestion automatique de la mémoire (garbage collection), typage fort, gestion des exceptions
- **Écosystème riche** : Vaste bibliothèque standard et écosystème de frameworks (Spring, Hibernate, etc.)
- **Performance** : JIT compilation, optimisations de la JVM
- **Sécurité** : Sandboxing, gestion des permissions, bytecode vérifié

## Structure de la formation

Cette formation couvre :

1. **Versions Java** : Java 8, 11, 17, 21, 25 et leurs features principales
2. **Frameworks** : Spring et Spring Boot
3. **Caching** : Redis, EhCache, Caffeine
4. **Resilience** : Resilience4J
5. **Observabilité** : OpenTelemetry
6. **Infrastructure** : JVM, Tomcat, Apache
7. **Documentation** : Swagger, OpenAPI

## Prérequis

- Connaissance de base en programmation orientée objet
- Familiarité avec les concepts de programmation (variables, fonctions, classes)
- Expérience avec un IDE (IntelliJ IDEA, Eclipse, ou VS Code)

## Objectifs d'apprentissage

À la fin de cette formation, vous serez capable de :

- Maîtriser les features des différentes versions de Java
- Développer des applications avec Spring et Spring Boot
- Implémenter des stratégies de caching efficaces
- Créer des applications résilientes avec Resilience4J
- Instrumenter vos applications avec OpenTelemetry
- Déployer et configurer des serveurs d'applications
- Documenter vos APIs avec Swagger et OpenAPI
