# Java 8 - Les Features Principales

Java 8, publié en mars 2014, a introduit des changements majeurs dans le langage, notamment la programmation fonctionnelle avec les expressions lambda et l'API Streams.

## Vue d'ensemble

Java 8 a révolutionné la façon d'écrire du code Java en introduisant :

- **Expressions Lambda** : Permettent d'écrire du code plus concis et fonctionnel
- **API Streams** : Traitement de collections de manière déclarative
- **Optional** : Gestion élégante des valeurs nulles
- **Méthodes par défaut dans les interfaces** : Évolution des interfaces sans casser le code existant
- **Nouvelle API Date/Time** : Remplacement de l'ancienne API Date/Calendar
- **CompletableFuture** : Programmation asynchrone améliorée
- **Nashorn** : Moteur JavaScript intégré
