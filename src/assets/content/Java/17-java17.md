# Java 17 - LTS et Features Principales

Java 17, publié en septembre 2021, est la troisième version LTS majeure. Elle apporte des features modernes et des améliorations de performance.

## Vue d'ensemble

Java 17 introduit :

- **Sealed Classes** : Contrôle de l'héritage
- **Pattern Matching** : Amélioration du switch et des instanceof
- **Records** : Classes de données immutables
- **Text Blocks** : Chaînes multi-lignes améliorées
- **Améliorations de performance** : Optimisations de la JVM
- **Nouveaux garbage collectors** : ZGC et Shenandoah
