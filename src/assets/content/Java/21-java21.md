# Java 21 - LTS et Features Principales

Java 21, publié en septembre 2023, est la quatrième version LTS. Elle apporte des features modernes et des améliorations significatives.

## Vue d'ensemble

Java 21 introduit :

- **Virtual Threads** : Threads légers pour la programmation concurrente
- **Pattern Matching amélioré** : Switch patterns plus puissants
- **String Templates** : Interpolation de chaînes
- **Sequenced Collections** : Collections avec ordre garanti
- **Améliorations de performance** : Optimisations continues
