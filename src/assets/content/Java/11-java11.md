# Java 11 - LTS et Features Principales

Java 11, publié en septembre 2018, est la deuxième version LTS (Long Term Support) après Java 8. Elle apporte des améliorations significatives.

## Vue d'ensemble

Java 11 introduit :

- **Mot-clé `var`** : Inférence de type pour les variables locales
- **Nouvelles méthodes String** : `isBlank()`, `lines()`, `strip()`, etc.
- **HTTP Client** : Client HTTP moderne et asynchrone
- **Modules** : Système de modules (Project Jigsaw)
- **Améliorations de performance** : Garbage Collector amélioré
- **Suppression de fonctionnalités** : Dépréciation de certains modules
