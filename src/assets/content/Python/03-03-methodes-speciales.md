# 6. Méthodes spéciales (dunder methods)

Les méthodes spéciales (appelées "dunder methods" pour double underscore) permettent de définir le comportement des objets Python.

## Vue d'ensemble

Cette section couvre les méthodes spéciales essentielles :

- `__str__` et `__repr__` : Représentation des objets
- `__eq__` et `__hash__` : Égalité et hachage
- `__len__` et `__iter__` : Collections et itération
- `__call__` : Objets appelables
- Surcharger intelligemment : Bonnes pratiques

## La magie de Python

Les méthodes spéciales sont ce qui rend Python si expressif et puissant. Elles permettent de créer des objets qui se comportent naturellement.
