# 5. Modèle objet Python

Le modèle objet de Python présente des particularités uniques, notamment l'héritage multiple et le duck typing.

## Vue d'ensemble

Cette section explore :

- Héritage multiple : Comment Python gère plusieurs parents
- MRO (Method Resolution Order) : L'ordre de résolution des méthodes
- Composition vs héritage : Quand utiliser chaque approche
- Duck typing : "Si ça marche comme un canard, c'est un canard"

## Philosophie Python

Python privilégie la flexibilité et la simplicité, ce qui se reflète dans son modèle objet unique.
