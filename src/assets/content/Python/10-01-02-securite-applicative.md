# Sécurité applicative

La sécurité au niveau applicatif concerne la protection contre les vulnérabilités courantes dans les applications web et les APIs.

## Vue d'ensemble

- Validation des entrées
- Injection (SQL, command, template)
- XSS / CSRF côté frameworks web
- Sérialisation JSON sûre
