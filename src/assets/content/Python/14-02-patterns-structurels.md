# Patterns structurels

Les patterns structurels concernent la composition de classes et d'objets pour former des structures plus grandes.

## Vue d'ensemble

Cette section explore :

- **Adapter** : Très courant pour harmoniser interfaces (API externes)
- **Decorator** : Python natif via fonctions ou classes. Très utilisé pour logging, validation, cache
- **Composite** : Pour arbres, fichiers, ou structures récursives

## Concepts clés

Python offre un support natif pour certains patterns structurels, notamment le Decorator qui est intégré au langage.
