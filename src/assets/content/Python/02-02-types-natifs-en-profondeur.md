# 2. Types natifs en profondeur

Python offre une riche collection de types natifs optimisés et puissants. Comprendre leurs caractéristiques, leurs performances et leurs cas d'usage est essentiel pour écrire du code efficace et Pythonic.

## Vue d'ensemble

Cette section explore en profondeur les types natifs Python :

- **int, float, bool** : Types numériques et booléens, leurs particularités et pièges
- **str** : Chaînes de caractères immutables, slicing, encoding Unicode
- **list, tuple, set, dict** : Collections Python, leurs caractéristiques et cas d'usage
- **Copies superficielles vs profondes** : Comprendre les références et éviter les bugs
- **Coût mémoire et performances** : Optimiser l'utilisation des types natifs

## Pourquoi c'est important

Les types natifs Python sont :
- **Optimisés** : Implémentés en C, très performants
- **Riches en fonctionnalités** : Méthodes intégrées puissantes
- **Pythonic** : Utilisés dans tout l'écosystème Python

Une compréhension approfondie permet de :
- Choisir le bon type pour chaque situation
- Éviter les pièges courants (références, copies)
- Optimiser les performances
- Écrire du code plus lisible et maintenable

## Structure de cette section

Chaque type natif sera exploré en détail avec :
- Caractéristiques fondamentales
- Opérations et méthodes disponibles
- Cas d'usage et bonnes pratiques
- Pièges courants et comment les éviter
- Exemples pratiques et exercices
