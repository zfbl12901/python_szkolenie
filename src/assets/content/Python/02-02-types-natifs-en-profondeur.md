# 2. Types natifs en profondeur

Python offre une riche collection de types natifs. Comprendre leurs caractéristiques, leurs performances et leurs cas d'usage est essentiel.

## Vue d'ensemble

Cette section explore en profondeur :

- int, float, bool
- str (immutabilité, slicing, encoding)
- list, tuple, set, dict
- Copies superficielles vs profondes
- Coût mémoire et performances

## Pourquoi c'est important

Les types natifs Python sont optimisés et offrent des fonctionnalités puissantes. Une compréhension approfondie permet d'écrire du code efficace et Pythonique.
