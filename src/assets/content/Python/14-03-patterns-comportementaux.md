# Patterns comportementaux

Les patterns comportementaux concernent la communication et la responsabilité entre objets.

## Vue d'ensemble

Cette section explore :

- **Observer / Pub-Sub** : Utile pour événements, notifications, GUI, frameworks web
- **Strategy** : Implémentation via classes ou fonctions (Python favorise fonctions first-class)
- **Command** : Pour scripts ou jobs programmables

## Concepts clés

Python favorise les fonctions first-class, ce qui simplifie l'implémentation de certains patterns comportementaux.
