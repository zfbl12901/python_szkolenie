# VIII. Python et data

Cette section explore l'utilisation de Python pour la manipulation de données, la visualisation et les bases du machine learning.

## Vue d'ensemble

Cette section couvre :

21. **Data manipulation** : Manipuler et analyser des données avec NumPy et Pandas
22. **Visualisation** : Créer des visualisations de données

## Pourquoi c'est important

Python est devenu le langage de référence pour la data science grâce à son écosystème riche. Comprendre ces outils est essentiel pour travailler avec des données.
