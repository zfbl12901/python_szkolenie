# Conteneurisation

La conteneurisation avec Docker permet de créer des environnements reproductibles et portables.

## Vue d'ensemble

Cette section explore :

- **Docker avec Python** : Créer des images Docker pour des applications Python
- **Images slim** : Optimiser la taille des images
- **Multi-stage builds** : Réduire la taille et améliorer la sécurité
- **Temps de build et cold start** : Optimiser les performances de build et de démarrage

## Concepts clés

Une bonne conteneurisation améliore la reproductibilité, la portabilité et facilite le déploiement.
