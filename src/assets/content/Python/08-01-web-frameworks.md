# 18. Web frameworks

Python offre plusieurs frameworks web, chacun avec ses forces et cas d'usage spécifiques.

## Vue d'ensemble

Cette section explore :

- Flask : Framework minimaliste et flexible
- FastAPI : Framework moderne pour APIs rapides
- Django : Framework full-stack complet
- Comparaison avec Spring Boot : Différences et similitudes

## Concepts clés

Choisir le bon framework dépend de vos besoins : simplicité, performance, fonctionnalités intégrées, ou écosystème.
