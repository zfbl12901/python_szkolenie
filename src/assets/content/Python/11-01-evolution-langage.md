# 26. Évolution du langage

Python évolue constamment. Comprendre comment suivre et comprendre ces évolutions est essentiel.

## Vue d'ensemble

Cette section explore :

- PEP (Python Enhancement Proposals) : Comment Python évolue
- Nouvelles versions : Suivre les nouvelles versions de Python
- Dépréciations : Comprendre et gérer les dépréciations

## Concepts clés

L'évolution de Python suit un processus structuré via les PEP, permettant à la communauté de participer et de comprendre les changements.
