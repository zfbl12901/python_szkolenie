# 29. Architecture Python

Concevoir des architectures Python robustes et maintenables est essentiel pour les projets de grande envergure.

## Vue d'ensemble

Cette section explore :

- Monolithes vs microservices : Choisir la bonne architecture
- Hexagonal / Clean architecture : Architectures modulaires et testables
- Organisation de projets : Structurer efficacement les projets Python

## Concepts clés

Une bonne architecture facilite la maintenance, les tests et l'évolution du code. Comprendre les différents patterns permet de faire les bons choix.
