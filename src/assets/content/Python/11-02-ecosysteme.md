# 27. Écosystème

L'écosystème Python évolue rapidement. Savoir identifier les tendances et distinguer les projets viables des projets morts est essentiel.

## Vue d'ensemble

Cette section explore :

- Frameworks émergents : Identifier les nouvelles tendances
- Libs maintenues vs mortes : Évaluer la viabilité des projets
- GitHub trends (avec esprit critique) : Analyser les tendances de manière critique

## Concepts clés

Tous les projets populaires ne sont pas nécessairement les meilleurs choix. Il faut évaluer la maintenance, la qualité et la pertinence.
