# II. Programmation orientée objet en Python

Python est un langage orienté objet qui permet de créer des classes et des objets. Cependant, le modèle objet de Python diffère de celui de langages comme Java ou C++.

## Vue d'ensemble

Cette section couvre les concepts fondamentaux de la POO en Python :

4. **Classes et objets** : Les bases de la création de classes en Python
5. **Modèle objet Python** : Les particularités du modèle objet Python
6. **Méthodes spéciales (dunder methods)** : La magie des méthodes spéciales

## Philosophie Python

Python adopte une approche pragmatique de la POO, privilégiant la simplicité et la flexibilité. Comprendre ces concepts est essentiel pour écrire du code Pythonique et exploiter pleinement les capacités du langage.
