# 16. Environnements

Les environnements virtuels permettent d'isoler les dépendances de chaque projet, évitant les conflits et facilitant la reproductibilité.

## Vue d'ensemble

Cette section couvre :

- `venv` : Environnements virtuels standard
- `pip` : Gestionnaire de packages Python
- `pipx` : Installation d'applications Python isolées
- `Poetry` : Gestion moderne de projets et dépendances
- `Conda` : Quand et pourquoi l'utiliser

## Concepts clés

Isoler les environnements est une pratique essentielle en Python pour éviter les conflits de versions et garantir la reproductibilité des projets.
