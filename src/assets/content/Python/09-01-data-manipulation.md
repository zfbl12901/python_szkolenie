# 21. Data manipulation

Manipuler efficacement les données est au cœur de la data science. NumPy et Pandas sont les outils essentiels pour cela.

## Vue d'ensemble

Cette section explore :

- NumPy : Calculs numériques et tableaux multidimensionnels
- Pandas : Manipulation de données structurées
- Vectorisation : Optimiser les performances avec la vectorisation
- Performance : Techniques d'optimisation pour les grandes données

## Concepts clés

Comprendre la vectorisation et les structures de données optimisées permet de traiter efficacement de grandes quantités de données.
