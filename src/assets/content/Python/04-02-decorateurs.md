# 8. Décorateurs

Les décorateurs sont l'un des concepts les plus puissants et élégants de Python, permettant de modifier le comportement des fonctions de manière non intrusive.

## Vue d'ensemble

Cette section explore :

- Principe : Comment fonctionnent les décorateurs
- Décorateurs avec paramètres : Décorateurs plus flexibles
- Cas d'usage réels : Logging, sécurité, cache
- Lisibilité vs magie noire : Trouver le bon équilibre

## Concepts clés

Les décorateurs exploitent le fait que les fonctions sont des objets de première classe en Python, permettant des patterns de design élégants et réutilisables.
