# 28. Culture Python

Comprendre la culture et la philosophie Python aide à écrire du code Pythonique et à s'intégrer dans la communauté.

## Vue d'ensemble

Cette section explore :

- Zen of Python : Les principes philosophiques de Python
- Communauté : S'engager dans la communauté Python
- Philosophie du langage : Comprendre l'approche Python

## Concepts clés

La culture Python privilégie la lisibilité, la simplicité et la communauté. Comprendre ces valeurs aide à mieux utiliser le langage.
