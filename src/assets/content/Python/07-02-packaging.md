# 17. Packaging

Créer et distribuer des packages Python permet de réutiliser du code et de partager vos créations avec la communauté.

## Vue d'ensemble

Cette section explore :

- `setup.py` : L'ancienne méthode de packaging
- `pyproject.toml` : La méthode moderne standardisée
- Semantic versioning : Gérer les versions de manière sémantique
- Distribution interne / open source : Distribuer vos packages

## Concepts clés

Comprendre le packaging Python est essentiel pour créer des bibliothèques réutilisables et les distribuer efficacement, que ce soit en interne ou en open source.
