# Environnements

La gestion des environnements et de la configuration est fondamentale pour des applications robustes en production.

## Vue d'ensemble

Cette section explore :

- **12-factor app** : Principes pour des applications cloud-native
- **Config par environnement** : Gérer différentes configurations selon l'environnement
- **Gestion des secrets** : Stocker et utiliser les secrets de manière sécurisée

## Concepts clés

Une bonne gestion des environnements permet de déployer la même application dans différents contextes (dev, staging, production) de manière fiable et sécurisée.
