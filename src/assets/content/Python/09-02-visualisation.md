# 22. Visualisation

La visualisation de données est essentielle pour comprendre et communiquer les insights. Python offre plusieurs bibliothèques puissantes.

## Vue d'ensemble

Cette section explore :

- Matplotlib : Bibliothèque de base pour la visualisation
- Seaborn : Visualisations statistiques élégantes
- Plotly : Visualisations interactives

## Concepts clés

Choisir la bonne bibliothèque dépend de vos besoins : statique vs interactif, simplicité vs contrôle, statistique vs généraliste.
