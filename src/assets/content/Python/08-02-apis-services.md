# 19. APIs et services

Créer des APIs robustes et maintenables nécessite de comprendre les bonnes pratiques et les outils disponibles.

## Vue d'ensemble

Cette section explore :

- REST : Principes et bonnes pratiques des APIs REST
- Validation (Pydantic) : Valider les données d'entrée
- Authentification : Sécuriser vos APIs
- Observabilité : Monitorer et déboguer vos services

## Concepts clés

Une API bien conçue est facile à utiliser, bien documentée, sécurisée et observable.
