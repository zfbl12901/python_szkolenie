# V. Performance et runtime

Cette section explore les aspects de performance et d'exécution de Python, essentiels pour développer des applications performantes et scalables.

## Vue d'ensemble

Cette section couvre :

13. **CPython et le GIL** : Comprendre le Global Interpreter Lock
14. **Concurrence et parallélisme** : Utiliser les threads et processus, asyncio vs threads, frameworks async
15. **Asynchrone** : Programmation asynchrone avec async/await
16. **Optimisation** : Profiling, benchmarking, structures de données, algorithmes
17. **Extensions et performance** : C extensions, Cython, NumPy, PyPy, offloading

## Pourquoi c'est important

Comprendre le runtime Python et les mécanismes de concurrence est crucial pour développer des applications performantes, surtout dans un contexte de traitement parallèle ou d'I/O intensif.
