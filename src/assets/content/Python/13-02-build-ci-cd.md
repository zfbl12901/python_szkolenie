# Build & CI/CD

L'intégration continue et le déploiement continu permettent d'automatiser la qualité et le déploiement.

## Vue d'ensemble

Cette section explore :

- **Linting** : Ruff, Black pour maintenir la qualité du code
- **Tests automatisés** : Intégrer les tests dans le pipeline CI/CD
- **Type checking en CI** : Vérifier les types automatiquement
- **Packaging en pipeline** : Automatiser la création et la publication de packages

## Concepts clés

Un pipeline CI/CD robuste garantit la qualité du code et facilite les déploiements fréquents et fiables.
