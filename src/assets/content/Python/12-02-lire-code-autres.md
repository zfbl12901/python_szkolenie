# 30. Lire le code des autres

Lire et comprendre le code de projets de qualité est l'une des meilleures façons d'améliorer ses compétences.

## Vue d'ensemble

Cette section explore :

- CPython : Comprendre l'implémentation de Python
- Frameworks open source : Analyser le code de frameworks populaires
- Apprendre par immersion : Techniques pour apprendre en lisant du code

## Concepts clés

Lire le code source de Python et des frameworks populaires permet de comprendre les patterns avancés et les meilleures pratiques.
